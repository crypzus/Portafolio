Curso Git & GitHub

Oracle Next Education 2023

Este curso tuvo un tiempo de duración de 8 horas donde te enseña control de versiones y colaboración en proyectos con Git y GitHub. Se aprende a realizar seguimiento de cambios, crear puntos de verificación y retroceder a versiones anteriores mediante ramificación. Git controla versiones localmente, mientras que GitHub almacena repositorios de forma remota para facilitar la colaboración, revisiones de código y gestión de problemas.

Este curso tubo un periodo de 50 horas donde aprendes a construir interfaces de usuarios interactivas y reactivas, una renderización mas eficiente usando un enfoque de virtual dom que permite utilizar las partes necesarias de la interfaz de usuario cuando los datos cambian mejorando así el rendimiento y la velocidad a través de aplicaciones en una sola página.
